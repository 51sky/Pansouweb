部署验证

在Linux服务器上部署时，请执行以下步骤：


# 1. 赋予部署脚本执行权限
chmod +x deploy-linux.sh

# 2. 执行部署（需要root权限）
sudo ./deploy-linux.sh

# 3. 或者使用Docker部署
docker-compose -f docker-compose.linux.yml up -d

# 4. 验证服务状态
curl http://localhost/health

网盘检测状态说明

搜索结果中将显示以下状态：

  • 🔵 检测中 - 正在检查链接有效性
  • 🟢 链接有效 - 资源可以正常访问
  • 🔴 资源失效 - 链接已失效或不存在
  • 🟡 需要提取码 - 链接有效但需要密码
  • ⚪ 未知状态 - 检测失败或无法判断
